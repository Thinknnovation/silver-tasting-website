# Silver Tasting

Luxury whiskey tasting website located in La Mesa, Cundinamarca, Colombia.

## Tech Stack

- Angular 20 Standalone
- SCSS
- Bootstrap 5
- GSAP Animations
- Mobile First

## Design Style

- Cinematic
- Dark Luxury
- Elegant
- Premium
- Warm lighting
- Awwwards inspired

## Main Sections

- Hero
- Experiences
- Events
- Gallery
- Testimonials
- Contact
- Footer

## UX Goals

- Premium experience
- Elegant animations
- Smooth scrolling
- Luxury branding
- Responsive design

## Notes

- Use standalone components
- Use SCSS modular architecture
- Avoid generic bootstrap look
- Prioritize cinematic UI
